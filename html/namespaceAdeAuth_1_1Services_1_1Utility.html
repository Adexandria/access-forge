class AdeAuth.Services.Utility.AccessOption ( AccessRule  accessRule = null)

Provides options for validating access rules and requirements.

The AccessOption class offers methods to validate passwords and email confirmation requirements based on specified access rules. It utilizes a set of validation rules to ensure compliance with security standards.

Parameters
accessRule

Validates whether the specified user's email confirmation requirement is met.

Template Parameters
TUserThe type of the user, which must inherit from ApplicationUser.
Parameters
userThe user whose email confirmation status is to be validated. Cannot be null.
Exceptions
ValidationExceptionThrown if the user's email confirmation requirement is not met.
