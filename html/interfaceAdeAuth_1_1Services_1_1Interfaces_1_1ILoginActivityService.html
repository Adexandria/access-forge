Detailed Description

Provides methods for managing login activities, including creation, update, deletion, and retrieval of login activity records.

This service interface defines both asynchronous and synchronous methods for handling login activities. It supports operations to create, update, delete, and fetch login activity records based on user identifiers and IP addresses.
