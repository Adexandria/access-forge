Detailed Description

Represents the result of an access operation, indicating success or failure and providing error details if applicable.

Represents the result of an access operation, including success status, data, and errors.

Use the static methods Failed and Success to create instances of this class.

This class extends AccessResult to include a data payload of type TModel . It provides static methods to create instances representing either success or failure of an access operation.

Template Parameters
TModelThe type of the data returned on a successful access operation.
