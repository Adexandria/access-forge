Detailed Description

Provides methods to validate various types of input data, such as text, passwords, emails, GUIDs, and integers.

The Validator class allows chaining of validation methods to accumulate errors, which can be checked by calling the Validate method. If any validation errors are present, a ValidationException is thrown. This class supports both default and custom validation logic through the use of an AccessOption.
