Detailed Description

Manages the user claim

This service allows for creating, updating, deleting, and fetching user claims. It operates on a DbSet<TEntity> of UserClaim entities within the provided database context.

Template Parameters
TDbContextThe type of the database context used by this service. Must be a subclass of DbContext.
Type Constraints
